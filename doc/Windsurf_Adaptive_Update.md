# แนะนำ "Adaptive": วิธีที่ชาญฉลาดกว่าในการใช้งาน Windsurf
 
 ![Adaptive update cover](/images/blogs/adaptive-hero.png)

 *วันที่เผยแพร่: 6 เมษายน 2026 | ใช้เวลาอ่านประมาณ 4 นาที*
 
 เรายินดีที่จะประกาศอัปเดตใหม่ล่าสุดสำหรับ **Windsurf** ซึ่งออกแบบมาเพื่อปรับปรุงประสบการณ์การใช้งานของคุณให้ดียิ่งขึ้น วันนี้เราได้เปิดตัวฟีเจอร์สำคัญหลายอย่าง ได้แก่ **Adaptive Model Router**, **หน้าต่างเลือกโมเดล (Model Picker) ดีไซน์ใหม่ที่มาพร้อมข้อมูลราคา**, และ **การยกเลิกขีดจำกัดการใช้งานรายวันสำหรับแพ็กเกจ Max**

ในช่วงที่ผ่านมา เราได้รับฟังเสียงตอบรับจากผู้ใช้งานอย่างใกล้ชิดว่า โครงสร้างราคาและโควต้าการใช้งานแบบใหม่นั้นมีความไม่ชัดเจน (Opaque) และในบางกรณีก็สร้างข้อจำกัดที่เข้มงวดเกินไปสำหรับการทำงานแบบมืออาชีพ ด้วยการอัปเดตในครั้งนี้ เรามุ่งมั่นที่จะแก้ไขปัญหาเหล่านั้นโดยตรง เพื่อมอบความโปร่งใสและอิสระในการเขียนโค้ดที่มากขึ้นให้กับนักพัฒนาทุกคน

 ## มีอะไรใหม่ในอัปเดตนี้บ้าง?
 
 ![Adaptive model picker](/images/blogs/adaptive-model-picker.png)

 ### 1. Adaptive Model Router (ระบบเลือกโมเดลอัจฉริยะ)
 เพื่อเพิ่มประสิทธิภาพการทำงานและบริหารจัดการโควต้าอย่างคุ้มค่าที่สุด เราได้นำเสนอ **Adaptive Model Router** ซึ่งเป็นระบบการจัดการคำสั่งอันชาญฉลาด ระบบนี้จะช่วยประเมินและสลับการใช้งานโมเดล AI ที่เหมาะสมที่สุดให้กับความซับซ้อนของ Prompt โดยอัตโนมัติ ทำให้คุณได้ผลลัพธ์ที่ดีที่สุดโดยไม่ต้องสิ้นเปลืองโควต้าระดับสูงไปกับคำสั่งง่ายๆ
 
 ### 2. ดีไซน์ Model Picker ใหม่ พร้อมแสดงบริบทด้านราคา (Pricing Context)
 ![Opus pricing card](/images/blogs/opus-pricing-card.png)

 ผู้ใช้หลายคนพบความลำบากในการคาดเดาว่าแต่ละการสนทนาใช้โควต้าไปเท่าไร เราจึงได้ออกแบบหน้าต่างเลือกโมเดล (Model Picker) ใหม่ทั้งหมด ตอนนี้คุณสามารถมองเห็น **บริบทด้านราคาและค่าใช้จ่ายของแต่ละโมเดล (Pricing Context)** ได้อย่างชัดเจนก่อนตัดสินใจเลือกใช้งาน 
 อินเทอร์เฟซใหม่นี้จะช่วยให้คุณเห็นภาพรวมและเข้าใจได้ทันทีว่าการเรียกใช้งานโมเดลแต่ละตัวจะส่งผลต่อขีดจำกัดการใช้งานโดยรวมของคุณอย่างไร ทำให้การบริหารจัดการโควต้าเป็นเรื่องง่ายและชัดเจนยิ่งขึ้น
 
 ### 3. ยกเลิกขีดจำกัดรายวัน (Daily Limits) สำหรับแพ็กเกจ Max
สำหรับผู้ใช้งานระดับ Power Users ที่สมัครแพ็กเกจ **Max** (ราคา $200) เราได้นำขีดจำกัดการใช้งานแบบรายวันออกไปเป็นที่เรียบร้อยแล้ว! 
เราเข้าใจดีว่ากระบวนการทำงานของนักพัฒนานั้นไม่ได้คงที่ทุกวัน บางวันคุณอาจจะพักผ่อน แต่บางวันคุณอาจจะต้องลุยสปรินต์อย่างหนัก การปลดล็อกครั้งนี้ทำให้ผู้ใช้แพ็กเกจ Max สามารถใช้โควต้าของตนเองได้อย่างอิสระตามรอบการทำงานจริง โดยไม่ต้องรอให้ถึงวันถัดไปเพื่อเริ่มใช้งานใหม่
